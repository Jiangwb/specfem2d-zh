simulation input parameters

title = Test

给这次的正演起个名字

SIMULATION_TYPE = 1

通过选择1和3，1表示计算正演的波场，3用于计算Adjoint Simulation,详情见manual中的Chapter 5

SAVE_FORWARD = .false.

保存正演的数据以便计算Adjoint Simulation，.false.表示不保存

NPROC = 8

正演计算中使用的核数，大于1需要基于MPI进行编译

ngnod = 9

网格化后定义一个计算单元的形状需要的节点数，对于二维情况可以取4(2×2)或者9(3×3)

NSTEP = 200

计算的步数，NSTEP×DT表示计算地震记录的长度

DT = 1.1d-3

表示计算的时间步长，这里是1.1×10-3s

time_stepping_scheme = 1

模拟中使用的积分方法是经典Newmark方法，有其相对应的计算稳定条件

AXISYM = .false.

二维的轴对称可以计算2.5D的情况，即水平成层。.false.表示2D

P_SV = .true.

模拟的结果是P_SV传播的结果，.false.对应SH波

GPU_MODE = .false.

使用GPU进行计算，设为.true.可以使用GPU，需要基于GPU进行编译

MODEL = default

根据Par_file中的设置项设定密度-速度模型

SAVE_MODEL = default

不保存模型信息

未确定作用的设置项有：

NOISE_TOMOGRAPHY = 0
partitioning_method = 3
setup_with_binary_database = 0
