receivers

台站的设置项

seismotype = 1

1表示记录位移，2表示记录速度，3表示记录加速度，4表示记录压强

subsamp_seismos = 1

表示降采样的倍数，一般设置为1，不降采样

NSTEP_BETWEEN_OUTPUT_SEISMOS = 1000

表示每1000步就保存一下地震记录到文件中，不建议使用太小的值。如果这个值大于NSTEP，则在正演结束后再保存记录

USER_T0 = 0.0d0

开始记录地震的起始时间，不使用SPECFEM2D自动计算的USER_T0

save_ASCII_seismograms = .true.

将正演计算输出的地震图保存为ASCII格式，就是OUTPUT_FILES里的*.semd，一列时间，一列振幅

save_binary_seismograms_single = .true.

将正演计算输出的地震图保存为binary格式，以单精度保留，对应OUTPUT_FILES中的Ux_file_single.bin

save_binary_seismograms_single = .false.

将正演计算输出的地震图保存为binary格式，以双精度保留，与单精度两个选项二选一

SU_FORMAT = .false.

将正演计算输出的地震图保存为seismic unix(即SU)格式，单精度。false表示不以此种方式保存

use_existing_STATIONS = .false.

true表示使用已经在DATA文件夹中存在的STATIONS文件，.false.表示读取Par_file中关于台站的设置项

nreceiversets = 2

控制测线的数量，对于每条测线上台站的数量、间隔等都要单独设置

nrec = 11

控制一条测线上台站的个数

xdeb = 300

第一个台站在X-Z坐标系下的X坐标

zdeb = 2200

第一个台站在X-Z坐标系下的Z坐标

xfin = 3700

最后一个台站在X-Z坐标系下的X坐标

zfin = 2200

最后一个台站在X-Z坐标系下的Z坐标 有了以上两个(x,z)坐标，控制测线头尾两端的位置，其余的台站等间隔地分布在这条测线上

record_at_surface_same_vertical = .true.

true表示所有的台站被强制设定在地表上，原来设定的Z值将不起作用。此时台站的Z值和地表高程有关。

未确定作用的设置项有：

USER_TRICK_FOR_BETTER_PRESSURE = .true.

anglerec = 0.d0
rec_normal_ro_surface = .false.
