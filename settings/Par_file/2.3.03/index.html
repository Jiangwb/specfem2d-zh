震源的设置项

这一部分的修改会涉及到SOURCE文件的相应修改，关于平面波作为初始震源的设置未详细了解

NSOURCES = 1

震源的个数

initialfield = .false.

false表示根据SOURCE文件中设定的震源参数作为震源，.true.表示使用内置的平面波作为震源

未确定作用的设置项有：

force_normal_to_surface = .false.

可能是产生地震的力是否垂直于地表

add_Bielak_conditions_bottom/right/top/left = .false.
ACOUSTIC_FORCING = .false.
