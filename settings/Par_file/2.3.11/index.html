输出图形文件

NSTEP_BETWEEN_OUTPUT_IMAGES = 100

每完成100步的计算就输出波场的图像信息(即快照，.jpg)

output_color_image = .true.

输出带有色彩的JPEG文件，显示波场信息

imagetype_JPEG = 2

波场信息中显示的物理量，2表示Z方向上的位移,1表示X方向上的位移，3表示总的位移 4对应X方向的速度分量，5对应Z方向的速度分量，6对应总的速度

factor_subsample_image = 1.0d0

控制输出图像的质量，大于1质量会下降，0-1之间质量会上升

DRAW_SOURCES_AND_RECEIVERS = .true.

控制图上是否会出现台站和震源的分布，.true.表示出现

DRWA_WATER_IN_BLUE = .true.

对于模型中的流体统一在图像中显示为蓝色

USE_SNAPSHOT_NUMBER_IN_FILENAME = .false.

控制snapshot的命名方式，false表示以NSTEP命名，true表示以顺序数字(1,2,3...)命名

output_postscript_snapshot = .false.

输出PS格式的图像，一般设置为.false.以节省计算时间与输出结果的存储空间

imagetype_postscript = 1

1对应位移，2对应速度，3对应加速度。会以小箭头的形式出现在PS文件中

meshvect = .true.

在PS文件上显示网格化

modelvect = .true.

在PS文件上显示模型，深色表示高速，.false.为不显示

boundvect = .true.

在PS文件上显示边界条件

subsamp_postscript = 1

类似于factor_subsample_image选项

为确定作用的设置项有：

cutsnaps = 1.
USE_CONSTANT_MAX_AMPLITUDE = .false.
CONSTANT_MAX_AMPLITUDE_TO_USE = 1.17d4
POWER_DISPLAY_COLOR = 0.30d0
interpol = .true.
pointsdisp = 6
sizemax_arrows = 1.0d0
US_LETTER = .false.
output_wavefield_dumps = .false.

可能是把波场信息输出成一个文本文件，不建议这样做，因为产生的数据量会很大，一般设置为false

imagetype_wavefield_dumps = 1
use_binary_for_wavefield_dumps = .false.
