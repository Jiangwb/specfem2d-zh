以EXAMPLES/initial_plane_wave_with_free_surface为例：

切换至该目录：

$ cd ./EXAMPLES/initial_plane_wave_with_free_surface

修改DATA文件夹中的Par_file文件，NPROC设为1，initialfield改为.false.

$ ./run_this_example.sh

程序计算的过程中会在屏幕上不断打印输出信息，等到最后可以输出：

see results in dictionary: OUTPUT_FILES/
done
运行结束的时间

说明SPECFEM2D已经编译成功。

进行计算过程，结束之后会生成OUTPUT_FILES文件夹，此文件夹内存储着计算结果

OUTPUT_FILES中的.semd为生成的地震记录，d为displacement，表示记录的是位移。借助脚本将其转化为sac可读的SAC文件，或是使用SAC的readtable命令也可以一个一个读取.semd。.jpg为位移场的snapshot，每100步会生成一个snapshot。
